# eCommerce App

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Características

- Listado de productos desde API
- Carrito de compras con gestión de cantidades
- Rutas dinámicas para detalle de productos
- Rutas protegidas (carrito requiere login)
- Autenticación básica
- Layout responsive
